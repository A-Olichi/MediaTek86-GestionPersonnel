# MediaTek86 - Gestion du personnel

Projet réalisé dans le cadre du BTS SIO SLAM.

## Contexte

Développement d'une application Windows Forms permettant la gestion du personnel, des services et des absences pour le réseau de médiathèques MediaTek86.

Technologies et outils utilisés :

• C#
• Windows Forms .NET Framework
• MySQL
• GitHub
• Architecture MVC


Compétences mobilisées :

• Programmation orientée objet (POO)
• Développement d'applications de bureau
• Conception de bases de données
• Gestion de versions avec GitHub
• Architecture MVC

## État du projet

Projet en cours de développement.
